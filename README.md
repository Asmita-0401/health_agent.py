# 🏋️ AI Health & Fitness Planner

An AI-powered Health & Fitness Planner built using **Python, Streamlit, Agno Agents, and Google Gemini API**. The application generates personalized diet plans, workout routines, and answers health-related questions based on the user's profile.

---

## 🚀 Features

- 🥗 Personalized Diet Plan
- 💪 AI Generated Fitness Plan
- 🤖 Interactive AI Health Assistant
- 📋 Daily Meal Recommendations
- 🏃 Customized Workout Routine
- 💬 Ask Questions About Your Plan
- 🎨 Modern Responsive Streamlit UI
- ⚡ Powered by Google Gemini AI

## ⚙ Installation

Clone the repository
```bash
git clone https://github.com/Asmita-0401/health_agent.py.git
```

Go to project folder

```bash
cd health_agent.py
```

Create virtual environment

```bash
python -m venv venv
```

Activate environment

Windows

```bash
venv\Scripts\activate
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
streamlit run health_agent.py
```

---

## 🔑 Environment Variables

Create a **.env** file.

```env
GEMINI_API_KEY=YOUR_API_KEY
```

Never upload your API Key to GitHub.

---

## 🎯 How it Works

1. Enter your Gemini API Key.
2. Fill your health profile.
3. Click **Generate Personalized Plan**.
4. AI creates a Diet Plan.
5. AI creates a Workout Plan.
6. Ask follow-up questions using the chatbot.
